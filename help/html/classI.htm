static reformat_role_string ( $  text) [static]

Add square brackets between list items.

This method is used to make searching for key-values in an SQL database work. Eg. PC,Mobile,Facebook would become: [PC], [Mobile],[Facebook]. You can then search for Like *[PC]* and not find 'PC' in the middle of a word. Method will also get rid of double square-bracket notation '[[' used in Impact plugins.

Parameters:
string | array$textThe string to parse.
Returns:
string Parsed text with square brackets.
