Add square brackets between list items.

This method is used to make searching for key-\/values in an SQL database work. Eg. PC,Mobile,Facebook would become: \mbox{[}PC\mbox{]}, \mbox{[}Mobile\mbox{]},\mbox{[}Facebook\mbox{]}. You can then search for Like $\ast$\mbox{[}PC\mbox{]}$\ast$ and not find 'PC' in the middle of a word. Method will also get rid of double square-\/bracket notation '\mbox{[}\mbox{[}' used in \hyperlink{namespaceImpact}{Impact} plugins.
